display: flex

เข้าใจ Flexbox
ด้วยตา ไม่ใช่ท่องจำ

Flexbox คือวิธีจัดเรียง "กล่อง" (item) ภายใน "กล่องแม่" (container) ด้านล่างเป็นสนามเด็กเล่นที่เพิ่ม/ลบกล่องได้ ตั้งค่าแต่ละกล่องแยกกันได้ และผสมคำสั่งของ container หลายตัวพร้อมกันได้จริง โค้ดจะอัปเดตให้ตรงกับสิ่งที่ลองเล่นเสมอ

โฟกัสวันนี้

justify-content — จัดของตาม "แกนหลัก"

แกนหลัก (main axis) คือทิศทางที่ item เรียงต่อกัน (ปกติคือแนวนอน) justify-content สั่งว่าจะดัน item ไปทางไหนของแกนนั้น และช่องว่างที่เหลือจะถูกจัดวางยังไง — กำหนดกล่อง .container (เส้นขอบสีฟ้า) ไว้ชัดเจน แล้วขึ้น "เลขระยะห่าง" จริงเป็น px ให้เห็นทุกช่อง เทียบกันทั้ง 6 แบบ

เส้นขอบ = ตัว container ที่กำหนดขนาดไว้ item ภายใน container เส้นวัดระยะ + ตัวเลข px จริงที่วัดได้

Flexbox Cheat Sheet

Propertyใส่ที่ค่าที่ใช้บ่อยทำหน้าที่
displaycontainerflexเปิดโหมด flexbox
flex-directioncontainerrow / columnกำหนดว่าแกนหลักไปทางไหน
flex-wrapcontainerwrap / nowrapให้ตัดขึ้นบรรทัดใหม่หรือไม่
justify-contentcontainercenter / space-betweenจัดตำแหน่งบนแกนหลัก
align-itemscontainercenter / stretchจัดตำแหน่งบนแกนขวาง
gapcontainer10pxระยะห่างระหว่าง item
flex-growitem0 / 1ขยายกินที่ว่างเท่าไหร่
flex-shrinkitem0 / 1ยอมยุบเล็กลงตอนพื้นที่ไม่พอไหม
flex-basisitemauto / 200pxขนาดตั้งต้นก่อนคำนวณ grow/shrink
align-selfitemcenteroverride align-items เฉพาะกล่องนี้
orderitem-1 / 1สลับลำดับการแสดงผลโดยไม่ต้องแก้ HTML

จำหลักง่าย ๆ: คุม container = จัดกลุ่ม, คุม item = คุมตัวเอง
